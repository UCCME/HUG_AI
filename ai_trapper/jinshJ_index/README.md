# SMC 聪明钱交易策略系统 (Smart Money Concepts Trading System)

本策略基于 SMC（Smart Money Concepts），核心是跟随机构资金足迹，自动识别结构破坏、机构订单区、价值缺口等形态，提供进场、止损、止盈参考。包含五大组件与多空组合逻辑。

## 1. 系统概述
聚焦机构订单流和市场结构，通过算法捕捉“聪明钱”痕迹，给出高胜率的入场位、支撑阻力和趋势反转提示。

## 2. 核心组件
- **订单块 OB**：机构挂单/进场区，强支撑阻力。红色=买入区（支撑），绿色=卖出区（阻力）。
- **BOS（Break of Structure）**：结构破坏，顺势延续信号。红色=突破前高，多头延续；绿色=跌破前低，空头延续。
- **CH（Change of Character）**：性格改变，趋势反转信号。红色=震荡转多；灰/绿=多转空。
- **FVG（Fair Value Gap）**：价值缺口/失衡区，价格倾向回补。红色缺口偏多，绿色缺口偏空。
- **EQH/EQL（Equal Highs/Lows）**：等高/等低，短线支撑阻力与流动性聚集区，常伴随假突破/反转。

## 3. 交易流程
**多头：**
1) 等待 CH（红）：下跌结束迹象  
2) 确认 BOS（红）：上升结构成立  
3) 入场：回调至红色 OB 或回补红色 FVG  
4) 止损：放在红色 OB 或 EQL 下方  
5) 止盈：目标绿色 OB 或 EQH

**空头：**
1) 等待 CH（空）：上涨受阻迹象  
2) 确认 BOS（绿）：下跌结构延续  
3) 入场：反抽至绿色 OB 或回补绿色 FVG  
4) 止损：放在绿色 OB 或 EQH 上方  
5) 止盈：目标红色 OB 或 EQL

## 4. 风险提示
- 部分信号可能滞后/重绘，需收盘确认。
- 窄幅震荡易出现假信号，BOS/CH 频繁切换。
- 严格执行仓位和止损管理。
