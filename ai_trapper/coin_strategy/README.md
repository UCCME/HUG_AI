# 加密交易工具精简策略（工具在精不在多）

本策略基于“工具在精不在多”的核心理念：不同段位只需匹配最关键的工具与方法，避免信息过载。目标是用最少的工具覆盖最重要的决策场景，并形成可执行的流程。

## 1. 总体原则

- 少即是多：一个阶段只保留 1-3 个核心工具。
- 先决策后工具：先确认交易风格，再选择工具。
- 工具分层：新手以“集成型工具”为主，进阶用“专项工具”。
- 风险优先：任何策略都要先处理风险，再讨论收益。

## 2. 角色分层与工具配置

### A. 新手 / 小白（只用 1 个工具）

- 核心工具：Coinglass
- 使用目标：看情绪、看风险、看方向
- 关键指标：资金费率、爆仓量、清算热力图

操作方式：
1) 先看资金费率：
   - 费率极高 = 多头拥挤，追涨风险大
   - 费率极低 = 空头拥挤，反弹机会多
2) 再看爆仓量：
   - 高爆仓区 = 情绪过热或过冷
3) 配合简单价格结构（高低点）决定是否轻仓试单

### B. 进阶 / 短线交易者

- 工具组合：Coinglass + TradingView
- 使用目标：情绪确认 + 技术入场

入场框架：
1) Coinglass 判断情绪（费率 + 爆仓量）
2) TradingView 确认技术结构（趋势、区间、突破）
3) 只在“情绪信号 + 技术信号”共振时进场

### C. 中高阶 / 长线周期玩家

- 工具组合：Glassnode + TradingView
- 使用目标：判断周期顶底 + 长线布局

核心策略：
- Glassnode 用于判断大周期拐点
- TradingView 用于确认价格结构
- 只做方向正确的大趋势，避免日内频繁交易

### D. 研究型 / 跟庄与新项目

- 工具组合：Nansen 或 Arkham + DefiLlama
- 使用目标：追踪聪明钱 + 寻找生态热度

核心策略：
- Nansen / Arkham 观察巨鲸地址动向
- DefiLlama 观察链上 TVL 变化
- 追踪聪明钱进入的生态与项目

## 3. 策略模块化设计

### 模块 1：行情过滤器（只做高质量行情）

- 过滤条件：
  - 费率极端
  - 爆仓量放大
  - 市场情绪极端一致

### 模块 2：入场信号（共振逻辑）

短线入场条件：
- Coinglass 出现情绪极端信号
- TradingView 出现关键结构位
- 同向共振才执行

长线入场条件：
- Glassnode 周期信号确认底部
- 价格已稳定在关键支撑

### 模块 3：仓位控制

- 单笔仓位不超过总资金 10%（新手 5%）
- 连续亏损 3 次，强制停手 24 小时

### 模块 4：止损 / 止盈

- 止损：跌破关键结构位即止损
- 止盈：
  - 短线：风险收益比 >= 1:2
  - 长线：分批止盈，保留底仓

### 模块 5：风险事件过滤

- 宏观重要事件（美联储、CPI、监管政策）前减仓或观望
- 高波动时优先轻仓

## 4. 具体策略示例

### 示例 1：短线情绪反转

- 工具：Coinglass + TradingView
- 条件：
  - 资金费率极端负值
  - 爆仓量放大
  - 价格在关键支撑企稳
- 执行：轻仓多单，止损设在支撑下方

### 示例 2：长线周期布局

- 工具：Glassnode + TradingView
- 条件：
  - Glassnode 显示周期底部信号
  - 价格稳定在长期均线附近
- 执行：分批建仓，长线持有

### 示例 3：聪明钱跟踪

- 工具：Nansen + DefiLlama
- 条件：
  - 巨鲸地址持续买入
  - 生态 TVL 快速增长
- 执行：先小仓位试探，确认趋势后加仓

## 5. 交易流程（执行清单）

1) 确认交易类型：短线 / 长线 / 跟庄
2) 打开对应工具，检查核心指标
3) 等待“共振信号”出现
4) 设置仓位与止损
5) 进场执行
6) 监控风险事件
7) 达到目标后止盈或退出

## 6. 策略总结

- 工具要精简：新手只用 Coinglass 也能完成大部分决策。
- 策略要分层：不同阶段用不同工具，不要一次性全上。
- 共振才交易：情绪 + 技术 + 逻辑一致才出手。
- 风险第一：仓位控制和止损优先级高于盈利。

## 7. 代码与示例数据

脚本文件：
- `coin_strategy/strategy_engine.py`：核心策略引擎（情绪 + 技术共振）
- `coin_strategy/report.py`：交易日志统计摘要

示例数据：
- `coin_strategy/sample_price.csv`
- `coin_strategy/sample_sentiment.csv`

快速运行：

```bash
python coin_strategy/strategy_engine.py --price coin_strategy/sample_price.csv --sentiment coin_strategy/sample_sentiment.csv
python coin_strategy/report.py
```
