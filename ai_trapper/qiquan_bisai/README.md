# 期权比赛策略记录：纯纯的进攻型策略

这是一个典型的“基本面驱动 + 期权高盈亏比 + 利润滚仓”的进攻型策略。
核心思路是：用有限本金通过期权放大杠杆，在确认的宏观或产业趋势中，配合极其严格的资金管理（出金与滚仓）来实现高收益，同时守住底线。

## 1. 核心交易哲学

- 弱化技术分析：几乎不看传统K线指标（MACD、KDJ等），认为多为噪声。
- 基本面+消息面驱动：只抓“能落地的机会”，关注核心驱动因素（政策变化、地缘政治、供需错配、天气影响）。
- 工具定位：期权作为放大杠杆的工具，而非单纯对冲工具；利用不对称性（亏损有限、收益无限）博取爆发性收益。

## 2. 入场逻辑：寻找“共振”

只在趋势与消息共振时入场：

- 估值修复逻辑：如碳酸锂价格长期低位、产业亏损，存在反弹需求。
- 事件驱动逻辑：
  - 地缘/天气：几内亚雨季/政策影响铝土矿发运（做多氧化铝）。
  - 宏观政策：新矿产资源法实施（做多碳酸锂）、中美经贸会谈（做多金银/PTA）。
  - 替代需求：中东局势导致原油涨 -> 生物燃油需求 -> 棕榈油涨。

## 3. 资金管理（生存法则）

这是能把小资金滚到大资金且不亏回去的关键。

- 本金安全至上（出金策略）：
  - 账户大幅盈利后，立即提取本金和部分利润。
  - 例：氧化铝赚到42万时，提现22.8万；碳酸锂赚到80万时，提现40万。
  - 目的：将交易资金变成“无压力本金”（全是利润在跑），心态更稳。

- 风险底线前置：
  - 下单前明确这笔钱即使归零或亏损50%也能接受。
  - 绝不满仓梭哈，始终留有余地。

## 4. 期权移仓策略（进攻与防守）

趋势延续时，通过“移仓”锁定利润并保持进攻性：

- 选合约标准：
  - 期限：到期日 1 个月以上，避免时间价值衰减过快。
  - 虚实度：浅虚值（OTM），通常在三档以内。
  - 流动性：优先整数关口合约（如 70000、75000），便于进出。

- 移仓比例（30/70 法则）：
  - 不全仓移：30% 移至新的浅虚值合约（进攻），70% 留在实值/平值合约（防守）。

- 操作时机：
  - 标的波动平缓时操作，避免剧烈波动时追涨杀跌导致高溢价。

## 5. 止损与离场逻辑

- 时间止损：买入后一周行情未启动，直接离场，避免时间价值损耗。
- 逻辑证伪止损：基本面逻辑被破坏或出现极端反转（如跌停），果断斩仓。
- 反向波动容忍度：不因单日大阴线恐慌，观察次日确认是否反转。
- 主动止盈（逃顶）信号：
  - 情绪狂热：价格到达整数关口，市场情绪过于一致。
  - 动能衰竭：高开低走，涨停未果。
  - 风险事件临近：周末/节假日前（防范交易所提保或监管政策）。
  - IV 过高：隐含波动率极高（如 90%），期权价格过贵，性价比下降，选择平仓。

## 6. 策略总结

这是主观趋势交易的极致体现：

- 胜率：可能不高（早期长期亏损 + 时间止损）。
- 赔率：极高（通过期权抓住少数大行情）。
- 核心竞争力：精准捕捉基本面逻辑 + 极其自律的出金和移仓管理。

## 7. 代码与运行

已提供一个简化的策略模拟脚本：`qiquan_bisai/strategy_sim.py`。

- 无数据时会自动生成合成行情，用于演示策略框架。
- 可选输入行情 CSV 与事件 CSV（日期、方向、IV）。
- 输出：`qiquan_bisai/equity_curve.csv`、`qiquan_bisai/trade_log.csv`、`qiquan_bisai/summary.csv`。

示例运行：

```bash
python qiquan_bisai/strategy_sim.py
```
